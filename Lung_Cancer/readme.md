Project Overview:
This project focuses on analyzing lung cancer patient data using Microsoft Excel to identify key risk factors, survival patterns, and treatment effectiveness.
An interactive dashboard was built to provide clear, data-driven insights using advanced Excel features.

Objectives:
1.Analyze factors affecting lung cancer survival
2.Study impact of smoking and medical conditions
3.Identify trends across countries and demographics
4.Build an interactive dashboard for easy analysis

Tools & Technologies:
 1.Microsoft Excel
Excel Skills Used:
1.VLOOKUP, HLOOKUP
2.INDEX + MATCH
3.Pivot Tables
4.Conditional Formatting
5.SUMIF/COUNTIF
6.Charts & Graphs
7.Slicers
8.KPI Indicators

Dataset Features:
1.Patient ID
2.Age
3.Gender
4.Country
5.Cancer Stage
6.Smoking Status
7.Cholesterol
8.Hypertension
9.Asthma
10.Cirrhosis
11.Other Cancer
12.Treatment Type
13.Survival Status

Project Workflow:
1.Data Cleaning-
Handled missing values.
Structured dataset for analysis.
2.Data Analysis-
Used INDEX + MATCH & VLOOKUP for data retrieval.
Applied logical functions for condition-based insights.
3.Data Aggregation-
Created Pivot Tables to analyze:
Survival rate,
Smoking impact,
Treatment effectiveness.
4.Dashboard Development-
Built interactive dashboard using
Slicers (Country, Gender, Stage, Treatment, Smoking Status),
KPIs,
Multiple charts.

Key Insights:
1.Smoking significantly impacts survival rates.
2.Stage III shows highest survival among stages.
3.Radiation treatment methods works more effectively.
4.Passive smoking cases are notably high.
5.Poland country have greatest survival rate while Belgium and Bulgaria have low chances of survival rate.
6.Male patient have high chances of affected by lung cancer.
